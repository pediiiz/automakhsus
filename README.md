# Auto Makhsus

Production Next.js website for `automakhsus.com`, the parent brand website for the Auto Makhsus automotive ecosystem.

## Positioning

Auto Makhsus is the corporate ecosystem behind TehranSandali, MrSeat, TehranSeat, Tuduzi, and ANI2203. It connects automotive interiors, car seats, upholstery, VIP van and camper interiors, multi-brand automotive service, detailing, options, and digital CRM/customer platforms.

## Pages

- Homepage
- Brands
- Services
- Projects / Ecosystem
- About
- Contact
- Cooperation / Partnership
- Digital Platform

## CRM

Lead forms write into the existing TehranSandali CRM database with:

- `source = automakhsus`
- `sourcePath = automakhsus`
- `sourcePage = current page`

Secrets must be stored only in the server-side environment file and never committed to Git.
