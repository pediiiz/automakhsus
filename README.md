# Auto Makhsus

Production Next.js website for `automakhsus.com`, the parent brand and technical/commerce hub for the Auto Makhsus automotive ecosystem.

## Positioning

Auto Makhsus is the parent brand, foreign/imported-car technical hub, marketplace/store strategy, and digital customer platform behind the main brands TehranSandali, ANI2203, and Tuduzi.

MrSeat and TehranSeat are TehranSandali sub-brands, not parent-level Auto Makhsus brands.

Auto Makhsus covers technical services, diagnostics, electrical/electronics, options, detailing, bodywork, paint, pre-purchase inspection, parts commerce, buy + install workflows, and CRM-backed customer operations for foreign/imported cars.

## CRM

Lead forms write into the existing TehranSandali CRM database with:

- `source = automakhsus`
- `sourcePath = automakhsus`
- `sourcePage = current page`
- `businessUnit = AutoMakhsus Technical` or `AutoMakhsus Marketplace` in conversion metadata/message context

Secrets must be stored only in the server-side environment file and never committed to Git.
